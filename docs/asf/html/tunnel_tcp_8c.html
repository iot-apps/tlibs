Exemplo Logger Wifi.

Telit / GS2101M wifi / EVB3 / Arduino M0 

Este e um exemplo de um sistema de logger de Tasks continuas (ASF/FreeRTOS) baseado em wifi

que repassa o trafego de uma das portas seriais para um socket aberto com um servidor local Permite tambem o fluxo de dados servidor->dispositivo, refletido na porta serial. Um tunel de porta serial por TCP.
