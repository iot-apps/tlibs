Exemplo logger Bluetooth.

Telit BlueMod+S42
EVK2
Arduino M0

Este e um exemplo de um sistema de logger baseado em uma Task (ASF) que repassa o trafego de uma das portas seriais para um socket aberto com um servidor remoto e bufferiza os bytes que sao recebidos enquanto a conexao nao estava ativa.
