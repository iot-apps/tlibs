Descrição detalhada

Exemplo logger Bluetooth Wifi.

BlueMode+S42 / GS2101M telit module
EVB3
BlueEva+S
Arduino M0

Este e um exemplo de um sistema de logger baseado em uma Task (ASF) que repassa o trafego de uma conexao de portas seriais bluetooth para um socket aberto com um servidor remoto e bufferiza os bytes que sao recebidos enquanto a conexao nao estava ativa.
