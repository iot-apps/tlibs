INSTALL:

Instala o python


Instala oo PIP:
	Baixa o script get-pip.py (https://bootstrap.pypa.io/get-pip.py)

	$python get-pip.py

Instala o MkDocs:
	$pip install mkdocs

Instala o mkdocs-material:
	$pip install mkdocs-material

Entra na pasta do wiki:
	$cd telit-libs

Roda servidor:
	$mkdocs serve

Acessa wiki no endereço:
	http://localhost:8000
